JavaScript Problem Set

Tasks to Accomplish

Read the Instructions file and accomplish the following tasks.

  1. Basics: Reaching into the page
    1. Reach into this page with JavaScript and change the *content* of the following tag to display your name: The Following Tag (id="name").
    2. Reach into this page with JavaScript and add a class to this special list item that changes its appearance (you'll need to add CSS for the class in question) (id="special")
    3. Reach into this page with JavaScript and add a style rule directly to this element with JavaScript. (id="style-me")
    4. Use JavaScript to go through every list item in the document and add a style rule (maybe change the text color -- i.e. element.style.color = '#777')
  2. Reacting to the User
    1. Make change colors when the user clicks it.

    2. Make this paragraph toggles back and forth between italics and not when you click on it.

    3. Create an obnoxious pop-up div that follows the mouse everywhere the user moves it on the page.
      Annoy Me!
    4. Write code which makes it so that every nested list-item on this page changes style when you click on it (you can select nested list items by using the selector "li li")
  3. Crawling the DOM
    1. Make the dialog divs below close when you click the "close" button on them.
      This should be a closable box.
      This should also be a closable box.
    2. Make it so that clicking on any list item makes *that* list item turn grey and then makes the *next* list item get highlighted (by highlighting it in yellow or bolding it or whatever floats your boat).
